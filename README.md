## ✨ Features

- 🎨 Beautiful ASCII art generation with **figlet**
- 🌈 Colorful text output with **lolcat**
- 💻 Terminal customization and styling
- 🚀 Easy to setup and use
- 🔧 Interactive user input
- ⚡ Lightweight and fast

---

## 📦 Requirements

This project requires:
- **Termux** (Android terminal application)
- **Python 3.x** or Python 2.x
- **Git** (for cloning the repository)

## 🔧 Installation

### Quick Start

```bash
rm -rf STYLISH-TERMUX
apt update && apt upgrade -y
pkg install git
pkg install python
git clone --depth=1 https://github.com/SP1D3R-X/STYLISH-TERMUX.git
cd STYLISH-TERMUX
python Termux.py
```

### Step by Step

1. **Update and upgrade packages:**
   ```bash
   apt update && apt upgrade -y
   ```

2. **Install required tools:**
   ```bash
   pkg install git
   pkg install python
   ```

3. **Clone the repository:**
   ```bash
   git clone https://github.com/SP1D3R-X/STYLISH-TERMUX.git
   cd STYLISH-TERMUX
   ```

4. **Run the script:**
   ```bash
   python Termux.py
   ```

---

## 🚀 Usage

Simply run the Python script and follow the on-screen prompts:

```bash
python Termux.py
```

The script will:
- Update your system packages
- Install necessary dependencies (figlet, ncurses-utils, ruby, lolcat)
- Ask for your name
- Display a stylish welcome message
